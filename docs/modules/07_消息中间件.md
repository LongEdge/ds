## 🔌 Kafka 模块

### kafka_cli.py — 生产者

```python
from kafka import KafkaProducer

BROKER = "<PRIVATE_HOST>:9002"
TOPIC = "<REDACTED_TOPIC>"

def send_message(msgs):
    producer = KafkaProducer(
        bootstrap_servers=BROKER,
        value_serializer=lambda v: json.dumps(v).encode('utf-8'),
        retries=3,
    )
    for msg in msgs:
        producer.send(TOPIC, msg)
    producer.flush()
    producer.close()
```

**问题**：
- ❌ 消息 key 未设置（无法保证顺序）
- ❌ 无分区策略
- ❌ 无 ACK 确认机制（`acks=1` 默认）
- ❌ 每次调用都新建/关闭 Producer（开销大）

## 🔄 通信模式对比

| 特性 | Kafka | MQTT |
|-----|-------|------|
| 吞吐量 | 高 | 中 |
| 适用场景 | 批量日志、消息队列 | 设备心跳、实时事件 |
| QoS | At-least-once / Exactly-once | At-most-once / At-least-once / Exactly-once |
| 持久化 | 支持消息持久化 | 依赖 client_id 离线队列 |
| 连接开销 | 较高 | 低 |
| 典型用途 | 任务日志收集 | 节点状态变更通知 |
