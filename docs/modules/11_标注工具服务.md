## 🔧 核心 API

### 皱纹相关

| API | 功能 |
|-----|------|
| `getWrinkleType` | 获取支持的皱纹类型列表 |
| `getImgDirLists` | 获取图片目录列表（按 group） |
| `getImgFileLists` | 获取图片文件列表（带标注状态） |

| `saveImgBase64ToMinio` | 保存标注遮罩到存储 |
| `deleteImgFromMinio` | 删除指定图片 |

### 图像处理相关

| API | 功能 |
|-----|------|

| `getCacheImgBase64FromMinio` | 获取图片（带缓存） |

## 🖼️ 图像缓存机制

```python
class CProcessorAPIWrinkle:
    imgBase64Cache = []  # 最多缓存 6000 条
    img_zoom_ratio = 0.75  # 图像缩放比例 (传前端时压缩)
    
    def saveImgBase64ToCache(self, img_path, img_base64_obj):
        # LRU 淘汰策略
        if len(self.imgBase64Cache) >= self.imgBase64CacheSize:
            self.imgBase64Cache.pop(0)
        self.imgBase64Cache.append({img_path_key: img_base64_obj})
```

**缓存策略**：
- 读取 MinIO 后缩放 0.75x 上传
- 保存时放大 1/0.75 = 1.33x 存回 MinIO
- **防止标注工具加载过大的图片**

## ⚠️ 问题 & 限制

| 问题 | 影响 |
|-----|------|
| 标注工具简陋（HTML 画布） | 复杂标注效率低 |
| 无版本控制 | 误操作无法回滚 |
| 权限控制已注释 | 当前环境无权限隔离 |
| 缓存无持久化 | 重启服务缓存丢失 |
| MinIO 独占存储 | 不支持其他存储后端 |
